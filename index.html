Google Analytics Client¶

Google Analytics Client is a wrapper for the google-api-python-client package to make using Google Analytics easier. The library automatically downloads more result pages if available, and converts string data into Python types.

The main interface is GoogleAnalyticsClient, the file sample.py contains a working example.

class gaclient.GoogleAnalyticsClient(service, profile, dimensions, metrics, logger=None, filters=None, limit=10000, sort=None, request_type=None)¶

Google Analytics Client.

Parameters:
  • service – An initialized Google Analytics service object, are a callable that returns one.
  • profile – Google Analytics profile id.
  • dimensions – A list of dimensions to download.
  • metrics – A list of metrics to download.
  • logger – Optional logger to use, if set to None no messages will be logged.
  • filters – Optional list of filter predicates.
  • limit – The maximum number of results per request, defaults to 10000 which is GAs maximum.
  • sort – Optional list of sort keys.
  • request_type – The class to use for GA requests, defaults to gaclient.Request.
download_date_range(start_date=None, end_date=None, days_per_request=1, limit=None, sort=None, auto_advance=True)¶

Download a range of data between start_date and end_date (inclusive).

Parameters:
  • start_date – Optional date of first record to download.
  • end_date – Optional date of last record to download.
  • days_per_request – Number of days to download in each request, If the resultset is too large GA will return sampled data, lowering days_per_request might prevent this behaviour. Default: 1.
  • limit – Optional limit for this request, defaults to self.limit.
  • sort – Optional sort predicate, defaults to self.sort.
  • auto_advance – If True requests will be issued until no more resulst are available, even is the result set is larger then limit results.
Returns:

A generator over all records in this result. The records are returned as collections.namedtuple instances.

start_date and end_date both default to yesterday, and accept either datetime.date or datetime.datetime or a string formatted as ‘yyyy-mm-dd` as input.

Request¶

class gaclient.Request(service, profile, start_date, end_date, dimensions, metrics, filters, sort, limit, auto_advance, logger=<logging.Logger object at 0x1063cd510>)¶
A wrapper for Google Analytics requests that supports
auto-advancing. The wrapper will continue to send requests as long as more data is available and auto_advance is True.
Parameters:
  • service – An initialized Google Analytics service object.
  • profile – GA profile id.
  • start_date – Date of first record retrieved.
  • end_date – Date of last record retrieved.
  • dimensions – A list of dimenions.
  • metrics – A list of metrics.
  • filters – List of filters or None.
  • sort – Sort predicate or None.
  • limit – Max number of results per request, of auto_advance is True then requests will be made until no more results are available.
  • auto_advance – see limit.

Note: objects of this type are lazy generators over their entire resultset.

records()¶

Returns a generator over all records.
